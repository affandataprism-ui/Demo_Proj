## Demo_Proj

Basic HTML demo project used for Git practice and testing.

- **Tech stack**: Plain HTML, CSS, and JavaScript (no build tools or frameworks).
- **Entry file**: `index.html`

### How to run

1. Open the project folder in your file explorer.
2. Double-click `index.html` to open it in your web browser.

You should see a simple demo page with a button that shows the time when clicked.
